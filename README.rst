==========
MRI Viewer
==========

A web application for visualizing and viewing MRI outputs


* Free software: MIT License


Installing
----------

Install the application

.. code-block:: console

    pip install -e .


Run the application

.. code-block:: console

    mri-viewer

Features
--------

* Basic code template
* Basic VTK rendering
* Temperature coloring
* Coordinate axes
* File/s picker
* Files list
* Data arrays list
* View modes list
